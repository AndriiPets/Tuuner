# Music_visualizer

music visualisation and video creation tool build with Next.js, Meyda.js and ffmpeg.wasm

## Getting Started

clone the repo

run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
